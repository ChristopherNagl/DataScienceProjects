# Machine Learning Drohnen Flugdaten

## Ziel
Vorhersage ob der hintere Rotor der Drohne beschädigt ist, oder nicht.

## Daten
Herkunft: Drohne DJI Spark
Aufzeichung: 4 verschiedene Flüge ohne Schäden sowie Flüge einen oder mehreren beschädigten Rotoren

## Ergebnis
Model als Pickle 
87,34% Genauigkeit
